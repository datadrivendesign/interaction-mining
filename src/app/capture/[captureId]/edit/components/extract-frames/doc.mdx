## Create interaction snapshots

Collect snapshots from your recording for each interaction with the app.

### Capturing snapshots
1. Start playing back your app recording.
2. Click the “Snapshot” button each time something happens — like a tap, swipe, or any interaction.
    - Avoid capturing animations or transitions.
3. Add a description to each snapshot to tell us what happened.
