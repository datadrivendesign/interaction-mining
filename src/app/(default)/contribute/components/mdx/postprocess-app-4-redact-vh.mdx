#### Redact View Hierarchy Metadata

The mobile app also allows invisible view hierarchy metadata redaction.

1. Select a screen from the screen list view to open the redaction interface.
2. Tap the FAB and List <ListCheck className="inline-block w-4 h-4 align-middle text-gray-400" /> button to expand a card view of the text and content description metadata of all view elements.
3. Tap an item card containing sensitive information in view hierarchy metadata.
4. Confirm redaction in the popup after tapping the the item card.

*Data redacted using the card view will only redact the view hierarchy metadata, not the visual screenshot itself.*
