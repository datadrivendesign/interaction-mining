## Redact Personal Data

Remove personal data from the screenshots to protect your privacy.

<video src="/demo.mp4" className="not-prose w-full h-auto rounded-lg focus:outline-hidden focus:ring-0" autoPlay loop playsInline muted></video>
### Tools
The redaction editor provides two tools to help you redact personal data.
- <Pencil className="inline-flex justify-center items-center size-6 p-1 mr-0.5 rounded-md text-white bg-black dark:text-black dark:bg-white aspect-square"/> Pencil —
**Drag** over sensitive information to redact it.
- <Eraser className="inline-flex justify-center items-center size-6 p-1 mr-0.5 rounded-md text-white bg-black dark:text-black dark:bg-white aspect-square"/> Eraser —
**Click/Drag** over a redaction to remove it.
